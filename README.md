# Script_Splitter
This is a program that extracts vital information of a film by its Final Draft script (xml format), separating it into fetchable assets such as Locations, Scenes, Characters, Props, and even DialogLines 

## Notes
The example script I included is one that I worked on in my Basics of Screenwriting class my freshman year of college, which won an award for Best Short at our local film festival!! It was lots of fun getting to revisit it and put it to further use.

As a filmmaker myself, I'm quite excited to be making this project, seeing as it will be very handy in practical settings with production workflow and organization. I'm fortunate enough to have access to Final Draft through my university, as well as plenty of past scripts to test the code on.
